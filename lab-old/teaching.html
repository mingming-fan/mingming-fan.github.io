HCIN 722: HCI with Mobile, Wearable, and Ubiquitous Devices

This class is about understanding and designing human-computer interactions with mobile, wearable, and ubiquitous devices. This class covers the following topics: Mobile Sensing; Mobile and Wearable Technologies for Accessibility; Computational Fabrication; Health Sensing; Infrastructure-mediate Sensing; and Attention and Emotion Sensing.

This is a seminar-style class with a focus on reading and discussing classic and state-of-the art research papers from top-tier conferences and journals in the field of mobile, wearable, and ubiquitous computing. This class also features a group project that involves designing and evaluating a mobile/wearable prototype.

Current semester: Spring 2021

Previous semesters: Spring 2020

HCIN 720: Prototyping Wearable and Internet of Things (IoT) Devices

This class is about prototyping physical artifacts that are connected to the Internet: allowing things sensed about the physical world to be acted on in the cloud, and to allow things happening on the Internet to be reflected in the physical world.

The focus of the class is on learning by doing (e.g., programming IoT devices to talk to each other and acquire sensor data from the environment and upload it to the internet).

Upcoming semester: Fall 2021

Previous semesters: Fall 2019, Fall 2020
